# Pagina di prova 

## Stili:  
- Grassetto: **prova**
- Italic: _prova_
- Barrato: 	~~prova~~

### Stili misti: 
- Grassetto e italic: ***prova _non prova_**
- Tutto grassetto e italico: ***prova***

## Altro: 
- Pedice: prova <sub>non<\sub> prova
- Apice: prova <sup>non<\sup> prova
- Sottolineato: <ins>prova<\ins>
