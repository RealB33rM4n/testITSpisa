## Stili:  
- Grassetto: **prova**
- Italic: _prova_
- Barrato: 	~~prova~~

### Stili misti: 
- Grassetto e italic: **prova _non prova_**
- Tutto grassetto e italico: ***prova***

## Altro: 
- Pedice: <sub>☝️</sub>🤓
- Apice: e<sup>x</sup>
- Sottolineato: <ins>prova</ins>
